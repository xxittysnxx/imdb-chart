An Analysis of IMDB Movie Genres from 1990-2022

Analyzing Modern Movie Datasets have always been a benchmark of data analysis and visualization. This websites demonstrate 3 graphs, highlighting different aspects of the raw data through graph engines powered by d3js.
DataVis Group 11 NCCU June, 2022


Genre Hierarchical Graph Chart

This is a hierarchical graph chart depicting the aggregated revenue of every genre. Drilling Down allows the user to sort by every half decade, giving the user a more holistic understanding of the trend and popularity of every genre, and finally every movie, as well as its profit and budget.

By using a bar chart race with scrubber to visualize the growth of every genre, the user is able to dynamically pin point the ranking changes as well as relative size in real time.

Genre Bar Chart Race

As we can see, drama has been the major revenue generator for movies in the early 1990's to early 2000's, but is soon surpassed by Action and Adventure Movies in the latter half, probably due to the heavy influx of Marvel BlockBusters

Rating/Revenue/Popularity Scatterplot

To be able to demonstrate 3 different variables (dimensions) on a 2d plane, a scatterplot is used with varying node sizes to depict the popularity accross an x/y (Revenue/Rating) plane. Selecting items in the legends allows the user to highlight different genres. In General, high rating movies have higher popularity, although with generally weak correlations, especially on less popular genres such as History and War.
